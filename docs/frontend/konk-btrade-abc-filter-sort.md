# Фильтр и сортировка по ABC для эндпоинтов konk-btrade (для фронтенда)

Параметры `abc` и `sortBy` поддерживаются в групповых эндпоинтах по конкуренту и производителю:

- **GET** `/api/analog-slices/konk-btrade/comparison-excel` — экспорт Excel сравнения срезов (остатки/цены)
- **GET** `/api/analog-slices/konk-btrade/sales-comparison-excel` — экспорт Excel сравнения продаж и выручки
- **GET** `/api/analog-slices/konk-btrade/sales-comparison` — агрегированные продажи/выручка по дням (JSON, для графиков)
- **GET** `/api/analog-slices/konk-btrade/stock-comparison` — агрегированные остатки по дням (JSON, для графиков)

## Параметр `abc` (опционально)

- **Значения:** `A` | `B` | `C` | `D` (одна буква).
- **Назначение:** фильтрация по наличию этой буквы в поле ABC артикула (без учёта регистра). В выборку попадают только артикулы, у которых в поле ABC есть указанная буква (например, при `abc=B` — артикулы с «101B», «50B» и т.п.).
- **Если не передан:** фильтр не применяется, в выборку попадают все артикулы группы по `konk` и `prod`.

## Параметр `sortBy` (опционально)

- **Значение:** `"abc"`.
- **Передавать можно с или без `abc`** — параметры независимы.

### Без `sortBy` (по умолчанию)

Блоки артикулов в документе/ответе отсортированы **по артикулу** (по возрастанию).

### При `sortBy=abc`

Блоки отсортированы:

1. Сначала по **букве ABC**: A, затем B, затем C, затем D.
2. Внутри каждой буквы — по **числовой части** значения ABC (например, 12, 56 для «12A» и «56A»).
3. При равенстве — по артикулу.

Артикулы без буквы A/B/C/D в поле ABC идут в конце.

**Пример порядка при `sortBy=abc`:**  
12A, 56A, 2B, 7B, 42B, 34C, 87C, 123C, 4D, 278D.

## Примеры query для фронтенда

- Сортировка по ABC без фильтра по букве (все артикулы группы, порядок A→B→C→D по числу):  
  `?konk=air&prod=gemar&dateFrom=2026-03-01&dateTo=2026-03-31&sortBy=abc`

- Только фильтр по букве B (без сортировки по ABC, порядок по артикулу):  
  `?konk=air&prod=gemar&dateFrom=2026-03-01&dateTo=2026-03-31&abc=B`

- Фильтр по B и сортировка по ABC (внутри B — по числу):  
  `?konk=air&prod=gemar&dateFrom=2026-03-01&dateTo=2026-03-31&abc=B&sortBy=abc`
