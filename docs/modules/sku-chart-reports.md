# Модуль Sku Chart Reports

## Назначение

JSON для графиков: сравнение агрегата SKU конкурента с Btrade (stock/sales), pie по производителям. Зависит от [sku-reporting](sku-reporting.md), [btrade-slices](btrade-slices.md), [arts](arts.md).

## Эндпоинты

Базовый путь `/api/sku-chart-reports`. См. [API sku-chart-reports](../api/sku-chart-reports.md).
